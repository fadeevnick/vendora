# Phase 00 — Local Infrastructure

## Goal

Establish that the local execution substrate for Vendora can be used as the basis for deeper runtime phases.

## Prerequisites

- runtime entry gate passed
- current execution asset identified: `../../vendora_codebase/`
- current implementation guide available

## `R1` Scope

- local services start
- main dependencies are reachable
- base developer bootstrap is usable

## `R2` Expansion

- repeatable reset/bootstrap flow
- stricter dependency health expectations
- better confidence in local reproducibility

## `R3` Expansion

- stronger failure/recovery rehearsal locally
- deeper operational confidence in dev substrate behavior

## Verification Shape

- service startup
- dependency reachability
- repeatability of local baseline

## Exit Criteria

- local infra is usable for the next runtime phase
- the chosen runtime depth for this phase is recorded in implementation tracking
